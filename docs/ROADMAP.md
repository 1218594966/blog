# 路线图

## 近期优先

1. 把主站作为统一入口继续打磨完成度。
2. 把博客从弹窗详情升级成独立文章页。
3. 给 `/tools`、`/projects`、`/lab` 逐步挂上第一个真实页面。
4. 把首页增加更明确的“探索入口”区域。

## 推荐新增功能

### 主站层

- 更清晰的博客详情页
- 文章搜索和标签过滤
- 阅读进度与目录导航
- 首页精选推荐位
- 简单的更新日志或动态卡片

### 工具层

- 提示词模板库
- 文本润色工具
- 文章摘要工具
- 常用 AI 链接导航
- 周报生成器

### 项目层

- AI 笔记本
- Prompt Atlas
- 工作流展示页
- 个人知识索引站

### 实验层

- 动效实验
- 对话交互试验
- 视觉风格尝试
- 快速原型

## 视觉和动效方向

- 让首屏加载、导航悬停、卡片进入和页面切换共享一套节奏。
- 把鼠标跟随效果收敛到重点区域，不在整站无差别开启。
- 增加更轻的分层视差，而不是堆更多动画。
- 给 AI 对话区增加流式输出和更细腻的反馈状态。
- 给工具中心、项目中心、实验区做统一但各有区别的视觉语言。

## 更长期的方向

- 让主站变成你所有作品和插件的总入口。
- 每个新功能优先考虑挂到 `/tools`、`/projects`、`/lab` 之一，而不是继续堆到首页。
- 当某个功能稳定后，再考虑独立成完整应用。
